# Output Formatting Skill

## Rules
- When asked to produce a report, follow the exact structure provided in the task.
- Output **only** the requested format (Markdown, JSON, etc.). Do not add extra reasoning or explanations.
- Stop generating once the required structure is complete.
- Prioritize clarity and professionalism in reports.
